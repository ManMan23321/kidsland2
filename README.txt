How to publish on GitHub Pages (free):
1. Create a GitHub account: https://github.com
2. Create a new repository named `kidsland` (or any name)
3. Upload all files and the assets folder to the repo root.
4. Go to Settings -> Pages -> Source -> main branch -> / (root) -> Save.
5. After a few minutes, your site will be live at: https://<your-username>.github.io/<repo-name>

How to connect a free domain via Freenom:
1. Go to https://www.freenom.com and register a free domain (.tk, .ml, .ga, .cf).
2. In Freenom domain settings -> Manage Domain -> Management Tools -> Nameservers -> Use custom nameservers:
   - Add GitHub Pages nameservers are not used; instead, use the GitHub Pages instructions: create a CNAME file with your domain and set DNS A records to GitHub IPs.
3. Follow GitHub Pages custom domain guide: https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site
(If you want, I will give you exact DNS records step-by-step for the domain you pick.)
